## Кодировки

ASCII - https://ru.wikipedia.org/wiki/ASCII - латиница кодом
WIN-1251 - https://ru.wikipedia.org/wiki/Windows-1251 - кирилица кодом
И т.д.
UTF8,16,32 - unicode
Ранее все символы влазили в 1 байт. Теперь 4 байта для всех символов.
UTF8 - 1-4 байта, компакта для ASCII - U+0000-U+10FFFF
UTF16 - 2 или 4 байта, быстрее в некоторых языках - U+0000-U+10FFFF
UTF32 - 4 байта, фиксированный размер, но неэффективна по памяти - U+0000-U+10FFFF
По первым символам unicode приложение может понять сколько примерно октетов использовано в символе и где его искать(как декодировать)

## Типы string и rune

Строки в Go - это неизменяемая последовательность байтов (byte = uint8)

// src/runtime/string.go
type stringStruct struct {
    str unsage.Pointer
    len int
}

s := "создать строку" - создать строку
var c byte = s[0] - получить доступ к байту(!) в строке
var s2 string = s[5:10] -= получить подстроку (в байтах!)
s2 := s + " again" - склеивать
l := len(s) - узнать длину в байтах

Unicode в Go 

Еоличество символов в строке не всегда равно длине строки

Символы Unicode в Go представлены с помощью типа rune = int32
Литералы рун выглядит так:
var r rune = ''
var r rune = '\n' - перенос строки
var r rune = '\xff' - последовательность байт
var r rune = '\u12e4' - unicode code-point
var r rune = '\'' - литерал руны

## Работа с текстами

Работа с UTF-8 в Go
Для удобной работы с Unicode и Utf-8 используем пакет unicode и unicode/utf8
DecodeRuneInString(s string) (r rune, size int) - получить первую руну
RuneCountInString(s string) (n int) - получить длину строки посимвольно
ValidString(s string)  bool - проверить валидность строки
Преобразование в слайс
Вы всегда можете преобразовать строку в слайс байтов или рун и работать далее со слайсом
s := "привет"
ba := []byte(s) - 24 байта
ra := []rune(s) - 6 байт
Итерация по строке
по байтам
for i: = 0; i < len(s); i++ {
    b:=s[i]
    i - строго последовательно
    b - имеет тип byte, uint8
}
по рунам
for i, r := range s{
    i - может перепрыгивать значения 1,2,4,6,9...
    r - имеет тип rune, int32
}
