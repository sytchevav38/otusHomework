## Структуры

Объявление пользовательских типов

type <Name> <Underlying Type>

Underlying Type может быть:
- Структурой - struct
- Простым типом - string, int, rune...
- Интерфейсом - interface
- Функцией - func
- Колллекцией - array, slice, map
- Каналом - chan

Псевдонимы типов

type <Alias> = <Type> // type chislo int
Знак "=" говорит о том, что это определение псевдонима

Стуктуры - фиксированный набор именованных переменных.
Переменные структуры называются полями.
Переменные размещаются радом в памяти.
struct{}  // пустая структура
type User struct{
    id int64
    Name string
    Age int
    friends []int64
}
