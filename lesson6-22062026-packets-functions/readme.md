## Функции
Функция - это независимый блок кода, который выполняет определённую задачу и может быть вызван из других частей программы.
### Параметры функции
1) Имя функции
2) Параметры
3) Тело функции
4) Результат (возвращаемое значени)
5) Вызов функции

### Виды функций
1) Функции без имены (анонимные функции или лямбды)
add := func(a int, b int) int {
    return a + b
}
2) Функции как значения
func test(a string){
    fmt.println(a)
}
test2=test
test2("Test123")
3) Замыкания
func counter() func() int {
    count:=0

    return func() int {
        count ++
        return count
    }
}
4) Вариативные функции
func sum(nums ...int) int {
    total :=0
    for _, num :=range nums {
        total += num
    }
    return total
}
5) Рекурсивные функции 
func factorial(n int) int {
    if n == 0{
        return 1
    }
    return n * factorial(n-1)
}
6) Функции с множественным возвращаемым значением
func divide(a, b int) (int,error) {
    if b == 0 {
        return 0, fmt.Errorf("division by zero")
    }
    return a / b, nil
}

## Функции дженерики

Дженерики (обобщённые типы) - это способ создания функций и типов, которые могут работать с раличными типами данных, сохраняя при этом типобезопасность.
func funcName[T any](value T) {
    fmt.Println(value)
}
func Swap[T1, T2 any](a T1, b T2) (T2, T1) {
    return b,a
}
