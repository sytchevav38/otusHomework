## Типизация в Go

Go - язык со статической (строгой) типизацией
1) Типы данных проверяются и на этапе компиляции, и на этапе выполнения.
2) Каждая переменная имеет не только значение, но и тип.
3) Как правило, типы не приводятся автоматически (неявно).

Объявление переменной
Переменные можно задаваться с помощью var
Константы создаются с помощью const
Можно использовать :=, чтобы сразу задать значение (short)

https://go.dev/ref/spec#Types

## Указатели

указатель - это адрес некоторого значения(ячейки) в памяти
Указатели строго типизированы
Zero Value для указателя - nil
Нельзя приводить указатели к челым числам
Если у указателя есть значение, то его можно разыменовать
var a = 10
var b *int <- указатель

### В стеке
Локальные переменные примитивных типов, объявленные внутри функции
Указатели на объекты (сами казатели хранятся в стеке, но объекты могут быть в куче)
### В куче
Динамически выделенные объекты через new и make
Переменные (или указатели), на которые есть указатели с более длительным временем жизни
(которое передаются за пределы функции или замыкаются в )
### В статической области
Все глобальные переменные и константы

## Приведение типов
