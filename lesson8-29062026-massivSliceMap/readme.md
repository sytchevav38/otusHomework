## Массивы
Это последовательность каких либо элементов
var arr [256]int //Массив цифр фиксированной длины
var arr [10][10]string //Многомерный массив размером 10 на 10
arr:=[...]int{1,2,3} // при создании массива автоматически считается размерность(длина)
arr:=[10]int{1,2,3,4,5} // первые 5 элементов 1-5, остальные 0

### Массивы: Операции 
v:=arr[1] - прочитать 2-ой элемент
arr[3]=1 - записать в 4-ый элемент 1
len(arr) - вывод длины массива 
arr[2:4] - получить срез массива с 3 по 5 элементы

## Слайсы
Это те же массивы, но переменной длины 
var s[]int - не инициализированный слайс, nil
s:= []int{} - с помощью литерала слайса
s:=make([]int,3) - с помощью функции make, s =={0,0,0}
s:=make([]int,3,10) - то же, но вручную управлем ёмкостью 3- длина, 10-ёмкость

## Словари (map)

Ключ - значение

var cache map[string]string
cache:=map[string]string{}
cache:=map[string]string{
    "one": "один",
    "two": "два",
    "three": "три",
}
cache:=make(map[string]string)
cache:=make(map[string]string,100)

value:= cache[key] - прочитать значение
value, ok:= cache[key] - прочитать значение и получить бул
_, ok := cache[key] - проверить наличие ключа
cache[key] = value - записать значение в инициализированный словарь
delete(cache, key) - удалить ключ из словаря, работает всегда
